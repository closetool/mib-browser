# mib-browser

Mib-Browser is a tui mib-browser

## Install

```shell
go get github.com/closetool/mib-browser
```

## Usage

```
Usage of ./mib-browser:
  -c string
        community (default "public")
  -h    help
  -i string
        ip (default "127.0.0.1")
  -p uint
        port (default 161)
```
